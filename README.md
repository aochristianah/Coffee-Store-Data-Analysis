# BrewVista-Store-Data-Analysis (Interactive Dashboard creation using MS Excel)
## Project Objective
The BrewVista Store wants analyze and visualize sales performance over time. So that the owner can understand how market, product type and size, location has contributed to sales performance across multiple years and how to grow more sales in subsequent years ahead.
